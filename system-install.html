系统安装/重装教程

学习如何安装和重装电脑系统，包括从U盘安装、正常系统内安装和系统镜像安装。

1

新硬盘从U盘安装系统

当电脑硬盘是新的或需要完全重装系统时，可以通过U盘启动来安装系统。以下是详细步骤：

步骤1：准备工作

  • 一个至少8GB的U盘
  • Windows或其他操作系统的ISO镜像文件
  • U盘启动盘制作工具（如Rufus、UltraISO等）
  • 一台可以正常使用的电脑

步骤2：制作U盘启动盘

  1. 下载并安装U盘启动盘制作工具
  2. 插入U盘，打开制作工具
  3. 选择下载好的系统ISO镜像文件
  4. 选择U盘作为目标设备
  5. 点击开始制作，等待完成

步骤3：设置电脑从U盘启动

不同厂家的主板进入BIOS/UEFI的按键不同，以下是常见品牌的按键：

品牌 进入BIOS/UEFI按键

技嘉（GIGABYTE）Del 或 F2
微星（MSI）Del 或 F2
映泰（BIOSTAR）Del
华擎（ASRock）Del 或 F2
英特尔（Intel）F2
戴尔（Dell）F2 或 F12
惠普（HP）F10 或 Esc + F10
联想（Lenovo）F2 或 F1

宏碁（Acer）F2 或 Del
神舟（HASEE）Del 或 F2
机械革命（MECHREVO）Del
雷神（ThundeRobot）Del

设置从U盘启动的步骤：

  1. 将制作好的U盘插入电脑
  2. 开机或重启电脑，快速按下对应的BIOS按键
  3. 进入BIOS/UEFI界面后，找到启动选项（Boot Options）
  4. 将U盘设置为第一启动项
  5. 保存设置并退出（通常按F10）

步骤4：安装系统

  1. 电脑重启后会从U盘启动，进入系统安装界面
  2. 选择语言、时间和键盘输入方法
  3. 点击"现在安装"
  4. 输入产品密钥（如果没有可以选择"我没有产品密钥"）
  5. 选择要安装的系统版本
  6. 接受许可条款
  7. 选择"自定义：仅安装Windows（高级）"
  8. 对硬盘进行分区（新硬盘需要先创建分区）
  9. 选择安装位置，点击"下一步"
  10. 等待系统安装完成，电脑会自动重启
  11. 完成后续设置（用户名、密码等）
2

正常开机时下载和安装系统

当电脑可以正常开机和进入系统时，可以直接在系统内下载和安装新版本的系统。

Windows系统更新安装

  1. 打开"设置"（Win + I）
  2. 点击"更新和安全"
  3. 点击"Windows 更新"
  4. 点击"检查更新"
  5. 如果有可用更新，点击"下载并安装"
  6. 等待更新下载完成，然后点击"立即重启"
  7. 电脑会自动完成更新安装

使用媒体创建工具升级

  1. 访问微软官方网站，下载Windows 10/11媒体创建工具
  2. 运行媒体创建工具
  3. 选择"立即升级这台电脑"，点击"下一步"
  4. 接受许可条款
  5. 选择要保留的内容（文件、设置和应用或仅文件）
  6. 点击"安装"，等待升级完成

Linux系统安装

对于Linux系统，可以通过以下方式安装：

  1. 访问Linux发行版官方网站，下载ISO镜像
  2. 使用虚拟机软件（如VirtualBox、VMware）直接安装
  3. 或使用WSL（Windows Subsystem for Linux）在Windows内安装
3

安装系统镜像

系统镜像是完整的系统安装文件，可以通过多种方式安装。以下是常见的安装方法：

推荐系统镜像下载链接

您可以从以下官方网站下载可靠的系统镜像：

  • 下载 Windows 11 镜像
  • 下载 Windows 10 镜像
  • 下载 Ubuntu 镜像
  • 下载 Debian 镜像

方法1：使用虚拟光驱安装

  1. 下载系统ISO镜像文件
  2. 右键点击ISO文件，选择"挂载"（Windows 10/11内置功能）
  3. 打开挂载的虚拟光驱，运行setup.exe
  4. 按照提示完成系统安装

方法2：使用第三方虚拟光驱软件

  1. 下载并安装虚拟光驱软件（如Daemon Tools、PowerISO等）
  2. 打开虚拟光驱软件，加载ISO镜像文件
  3. 打开虚拟光驱，运行安装程序
  4. 按照提示完成系统安装

方法3：使用PE系统安装

  1. 制作PE启动盘（如微PE、大白菜PE等）
  2. 将系统ISO镜像文件复制到PE启动盘中
  3. 从PE启动盘启动电脑
  4. 在PE系统中，使用安装工具（如CGI备份还原工具）安装系统镜像
  5. 选择系统镜像文件和安装位置
  6. 点击"安装"，等待完成后重启电脑

方法4：使用网络安装

某些Linux发行版支持网络安装，可以通过以下方式：

  1. 下载网络安装镜像或使用PXE启动
  2. 从网络安装镜像启动电脑
  3. 按照提示连接网络
  4. 选择要安装的软件包和组件
  5. 等待系统从网络下载并安装完成
4

如何分区（添加新的盘符，如D盘）

当您需要从现有硬盘中划分出新的盘符（如D盘）时，可以通过Windows系统自带的磁盘管理工具来完成。以下是详细步骤：

步骤1：打开磁盘管理工具

  1. 右键点击桌面上的"此电脑"图标
  2. 如果是Windows 11系统，需要先点击"显示更多选项"
  3. 在弹出的菜单中，点击"管理"选项
  4. 在计算机管理窗口中，点击左侧导航栏的"存储"，然后选择"磁盘管理"

步骤2：压缩现有卷

要创建新的盘符，首先需要从现有磁盘分区中压缩出一部分空间：

  1. 在磁盘管理窗口的下方图表区域，找到需要划分空间的磁盘分区（例如，从C盘提取空间，就找到"OS C:"）
  2. 右键点击该分区，选择"压缩卷"选项
  3. 系统会自动分析该分区的可用压缩空间，稍等片刻
  4. 在"压缩卷"对话框中，输入要压缩的空间大小（1000MB = 1GB）
  5. 点击"压缩"按钮，等待压缩过程完成

步骤3：创建新的分区

压缩完成后，会出现一个未分配的空间，现在可以将其创建为新的盘符：

  1. 在磁盘管理窗口中，找到刚刚压缩出来的未分配空间（显示为黑色）
  2. 右键点击该未分配空间，选择"新建简单卷"选项
  3. 进入"新建简单卷向导"，点击"下一步"
  4. 指定卷大小（默认为全部可用空间，可根据需要调整），点击"下一步"
  5. 分配驱动器号和路径，选择一个可用的驱动器号（如D、E、F等），点击"下一步"
  6. 格式化分区，选择文件系统（建议使用NTFS），分配单元大小保持默认，输入卷标（可选），勾选"执行快速格式化"，点击"下一步"
  7. 点击"完成"按钮，等待格式化完成

步骤4：验证新盘符

创建完成后，可以通过以下方式验证新盘符是否成功创建：

  1. 打开"此电脑"
  2. 在"设备和驱动器"区域，查看是否出现了新创建的盘符（如D盘）
  3. 可以右键点击新盘符，选择"属性"查看详细信息

注意事项

  • 压缩卷时，系统会自动计算最大可用压缩空间，建议不要压缩过多，保留足够空间给原分区
  • 格式化过程会清除分区上的所有数据，确保未分配空间中没有重要数据
  • 如果需要创建多个分区，可以重复上述步骤，从未分配空间中继续划分
  • 在Windows 11系统中，右键菜单可能与之前版本有所不同，需要先点击"显示更多选项"才能看到完整菜单
