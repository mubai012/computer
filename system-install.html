系统安装/重装教程

学习如何安装和重装电脑系统，包括从U盘安装、正常系统内安装和系统镜像安装。

1

新硬盘从U盘安装系统

当电脑硬盘是新的或需要完全重装系统时，可以通过U盘启动来安装系统。以下是详细步骤：

步骤1：准备工作

  • 一个至少8GB的U盘
  • Windows或其他操作系统的ISO镜像文件
  • U盘启动盘制作工具（如Rufus、UltraISO等）
  • 一台可以正常使用的电脑

步骤2：制作U盘启动盘

  1. 下载并安装U盘启动盘制作工具
  2. 插入U盘，打开制作工具
  3. 选择下载好的系统ISO镜像文件
  4. 选择U盘作为目标设备
  5. 点击开始制作，等待完成

步骤3：设置电脑从U盘启动

不同厂家的主板进入BIOS/UEFI的按键不同，以下是常见品牌的按键：

品牌 进入BIOS/UEFI按键
华硕（ASUS）Del 或 F2
技嘉（GIGABYTE）Del 或 F2
微星（MSI）Del 或 F2
映泰（BIOSTAR）Del
华擎（ASRock）Del 或 F2
英特尔（Intel）F2
戴尔（Dell）F2 或 F12
惠普（HP）F10 或 Esc + F10
联想（Lenovo）F2 或 F1
ThinkPadEnter + F1
宏碁（Acer）F2 或 Del
神舟（HASEE）Del 或 F2
机械革命（MECHREVO）Del
雷神（ThundeRobot）Del

设置从U盘启动的步骤：

  1. 将制作好的U盘插入电脑
  2. 开机或重启电脑，快速按下对应的BIOS按键
  3. 进入BIOS/UEFI界面后，找到启动选项（Boot Options）
  4. 将U盘设置为第一启动项
  5. 保存设置并退出（通常按F10）

步骤4：安装系统

  1. 电脑重启后会从U盘启动，进入系统安装界面
  2. 选择语言、时间和键盘输入方法
  3. 点击"现在安装"
  4. 输入产品密钥（如果没有可以选择"我没有产品密钥"）
  5. 选择要安装的系统版本
  6. 接受许可条款
  7. 选择"自定义：仅安装Windows（高级）"
  8. 对硬盘进行分区（新硬盘需要先创建分区）
  9. 选择安装位置，点击"下一步"
  10. 等待系统安装完成，电脑会自动重启
  11. 完成后续设置（用户名、密码等）
2

正常开机时下载和安装系统

当电脑可以正常开机和进入系统时，可以直接在系统内下载和安装新版本的系统。

Windows系统更新安装

  1. 打开"设置"（Win + I）
  2. 点击"更新和安全"
  3. 点击"Windows 更新"
  4. 点击"检查更新"
  5. 如果有可用更新，点击"下载并安装"
  6. 等待更新下载完成，然后点击"立即重启"
  7. 电脑会自动完成更新安装

使用媒体创建工具升级

  1. 访问微软官方网站，下载Windows 10/11媒体创建工具
  2. 运行媒体创建工具
  3. 选择"立即升级这台电脑"，点击"下一步"
  4. 接受许可条款
  5. 选择要保留的内容（文件、设置和应用或仅文件）
  6. 点击"安装"，等待升级完成

Linux系统安装

对于Linux系统，可以通过以下方式安装：

  1. 访问Linux发行版官方网站，下载ISO镜像
  2. 使用虚拟机软件（如VirtualBox、VMware）直接安装
  3. 或使用WSL（Windows Subsystem for Linux）在Windows内安装
3

安装系统镜像

系统镜像是完整的系统安装文件，可以通过多种方式安装。以下是常见的安装方法：

推荐系统镜像下载链接

您可以从以下官方网站下载可靠的系统镜像：

  • 下载 Windows 11 镜像
  • 下载 Windows 10 镜像
  • 下载 Ubuntu 镜像
  • 下载 Debian 镜像

方法1：使用虚拟光驱安装

  1. 下载系统ISO镜像文件
  2. 右键点击ISO文件，选择"挂载"（Windows 10/11内置功能）
  3. 打开挂载的虚拟光驱，运行setup.exe
  4. 按照提示完成系统安装

方法2：使用第三方虚拟光驱软件

  1. 下载并安装虚拟光驱软件（如Daemon Tools、PowerISO等）
  2. 打开虚拟光驱软件，加载ISO镜像文件
  3. 打开虚拟光驱，运行安装程序
  4. 按照提示完成系统安装

方法3：使用PE系统安装

  1. 制作PE启动盘（如微PE、大白菜PE等）
  2. 将系统ISO镜像文件复制到PE启动盘中
  3. 从PE启动盘启动电脑
  4. 在PE系统中，使用安装工具（如CGI备份还原工具）安装系统镜像
  5. 选择系统镜像文件和安装位置
  6. 点击"安装"，等待完成后重启电脑

方法4：使用网络安装

某些Linux发行版支持网络安装，可以通过以下方式：

  1. 下载网络安装镜像或使用PXE启动
  2. 从网络安装镜像启动电脑
  3. 按照提示连接网络
  4. 选择要安装的软件包和组件
  5. 等待系统从网络下载并安装完成
