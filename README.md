# 电脑维修教程网站

一个全面的电脑维修、系统安装、故障排除和优化教程网站，提供详细的步骤说明和实用技巧。

## 网站功能

### 1. 首页
- 网站介绍和主要内容导航
- 快速访问各个教程模块
- 响应式设计，支持移动端访问

### 2. 电脑安装教程
- 台式电脑安装步骤
- 笔记本电脑安装步骤
- 支持直接访问特定教程类型

### 3. 系统安装/重装
- 新硬盘从U盘安装系统
- 正常开机时下载和安装系统
- 安装系统镜像
- 硬盘分区教程（添加新盘符，如D盘）

### 4. 电脑优化
- 笔记本电脑清理方法
- 台式电脑清理方法
- Windows系统优化方法
- Mac系统优化方法
- Linux系统优化方法
- 显卡驱动安装教程（N卡和A卡）

### 5. 故障排除
- 电脑无法开机的解决方法
- 电脑蓝屏的解决方法
- 电脑黑屏的解决方法
- 电脑卡顿的解决方法
- 电脑自动关机的解决方法

### 6. 关于我们
- 网站介绍
- 更新日志

## 文件结构

```
电脑维修/

├── optimization.html   # 电脑优化教程
├── troubleshooting.html # 故障排除教程
├── about.html          # 关于我们
```

## 技术特点

1. **响应式设计**：适配各种屏幕尺寸，包括桌面端、平板和手机
2. **模块化结构**：每个教程都采用模块化设计，易于维护和扩展
3. **清晰的导航**：提供直观的导航菜单，方便用户快速访问所需内容
4. **详细的步骤说明**：每个教程都包含详细的步骤说明，适合不同水平的用户
5. **现代化UI**：采用现代化的设计风格，提供良好的用户体验
6. **移动端优化**：针对移动端设备进行了专门优化，包括汉堡菜单

## 如何使用

1. 在本地启动HTTP服务器：
   ```
   python -m http.server 8000
   ```

2. 在浏览器中访问：
   ```
   http://localhost:8000
   ```

3. 浏览各个教程页面，学习相关知识
